# oh-my-zsh-theme-vini4

Custom Oh My Zsh theme — Powerline-style prompt with git, venv, and tmux segments.

## Features

- Powerline-style segment separators
- Git branch + dirty/clean status (color-coded)
- Python virtualenv indicator
- Tmux session info in right prompt
- Fully customizable colors and segment ordering

## Install

```bash
./install.sh
```

## Symlinks

| Source | Target |
|--------|--------|
| `vini4.zsh-theme` | `~/.oh-my-zsh/custom/themes/vini4.zsh-theme` |

Based on Comfyline theme by imnotpua, customized with personal color palette.
